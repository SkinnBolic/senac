O HTML

O html é uma linguagem de marcação para internet.
É a linguagem base da internet necessaria para definir
onde cada elemento(texto, imagem, video, etc) deve estar.

O navegador(Chrome, IE, Firefox, Opera, Safari) "lê" o código
digitado e o "interpreta" exibindo assim na tela do site
já formatando e não códigos e TAGS.

O CSS

Folha de estilo em cascata, serve parar formatar as TAGSdefinidas mp HTML

As formatações incluem cor, alinhamento, altura, largura, aparência de objetos em geral.

PRINCIPAIS TAGS

  • A tag <p> serve para definir um parágrafo.
  • A tag <h1> serve para definir títulos(h1 a h6).
  • A tag <br> serve para quebrar uma linha simples.
  • A tag <ul> serve para criar uma linha não ordenada.
  • A tag <code> serve para marcar a cor selecionada
  • A tag <li> serve para marcar o inicio do parágrafo com bolinha
